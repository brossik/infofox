Переменные и их типы

Название Как писать в коде Хранит в себе Пример данных
Integer int Целые числа 3, 10, -2, 0
Float float Числа с дробной частью 10.23, -0.1, 3,1415
String str Строка символов hello, foxford, 2024
Boolean bool Хранит True либо False true, false
